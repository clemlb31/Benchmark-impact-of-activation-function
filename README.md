# Benchmark-impact-of-activation-function


Utilisation de 5 jeux de données pour évaluer l'influence des fonctions d'activations sur la performance des modèles

## Données

https://www.kaggle.com/datasets/umitka/food-price-inflation RNN, Regression

https://www.kaggle.com/datasets/minahilfatima12328/performance-trends-in-education Dense, Regression

https://www.kaggle.com/datasets/alessiocorrado99/animals10/data CNN, Classification

https://www.kaggle.com/datasets/sebastianwillmann/beverage-sales Dense, Regression

https://keras.io/api/datasets/mnist/ CNN, Classification
